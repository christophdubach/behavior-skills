# Barrierefreie Social Media: Emojis

Vor dem exzessiven Einsatz von Emojis sollte man sich bewusst sein, wie diese tatsächlich heissen und dass dieser Name vom Screenreader ausgesprochen wird. Ein Blick auf eine Emoji-Übersicht kann sehr aufschlussreich sein, z.B.:
- https://de.wikipedia.org/wiki/Unicodeblock_Zus%C3%A4tzliche_piktografische_Symbole
- https://allgemeinbildung.ch/cms/pages/a-z/listen/liste-emojis.php

Man realisiert dadurch hoffentlich schnell, wie stark die Verständlichkeit leidet, wenn mehrere Emojis hintereinander gesetzt werden – insbesondere innerhalb eines Satzes.

Zudem wird deutlich, dass Emojis häufig zweckentfremdet verwendet werden: Für Nutzende von Screenreadern ist vielleicht nicht sofort klar, dass der Beitragsverfassende eigentlich einen weihnachtlichen Stern gemeint hat, als er das Emoji «Schwindlig» eingesetzt hat, oder dass «Betende Hände» als «High Five» interpretiert werden sollen.

PS. Nicht alle Screenreader übersetzen den offiziellen englischen Unicode-Namen eines Emojis gleich, wie Jan Hellbusch in seinem Blog-Artikel aufzeigt: https://www.hellbusch.de/emojis/
