# UX Architect — Grounding Data

Domain knowledge and reference material for the `personas/ux-architect` skill.

---

## Core UX Principles

- Progressive disclosure — reveal complexity only when needed
- Reduce cognitive load — simplify decisions, minimize noise
- Clear information hierarchy — what matters most is seen first
- Consistency through patterns — reuse before inventing
- Accessibility and inclusiveness — design for the full range of users

---

## Common UX Patterns

- Step-by-step flows — guide users through complex tasks in stages
- Decision trees — map branching user paths and edge cases
- State machines — model UI states and transitions explicitly
- Error recovery flows — design the unhappy path, not just the happy one
